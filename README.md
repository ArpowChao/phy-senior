# 學測物理 SOIL 核心複習系統

以 108 課綱必修物理為主軸，整合知識節點複習、互動物理實驗、分級題庫、歷屆試題與命題統計的純前端學習網站。

## 線上使用

[開啟學測物理 SOIL 核心複習系統](https://arpowchao.github.io/phy-senior/)

網站不需要註冊或安裝套件，使用桌面或行動裝置的現代瀏覽器即可開啟。

[查看 GitHub 原始碼與下載專案](https://github.com/ArpowChao/phy-senior)

## 系統內容

### 7 大模組與 28 個知識節點

教材依課綱與解題關係拆分為 N01–N28，並收納於七個主題模組：

1. 尺度、單位與測量
2. 力與運動
3. 萬有引力與天體運動
4. 能量與熱學
5. 波動、光及聲音
6. 電磁現象與用電安全
7. 量子現象與基本交互作用

每個模組採用 SOIL 結構整理：

- **S — Summary / Formulas：**核心公式、物理量、單位與適用條件。
- **O — Overview / Concept Map：**觀念關聯與理論脈絡。
- **I — Insights / Key Features：**題幹特徵、模型判斷與常見陷阱。
- **L — Life Connections / Exams：**生活情境、科技應用與歷屆題連結。

### 互動物理實驗室

互動實驗可調整參數、暫停動畫並即時觀察物理量變化，也會嵌入對應的知識節點頁面。

- 天體運行：日心圓軌道、地心等價分解與行星逆行。
- 波動現象：惠更斯原理、反射、折射、干涉與繞射。
- 都卜勒效應：波源移動、觀察者移動及兩者同時移動，並比較視波長、視波速與視頻率。
- 光電效應：波長、輻照度、材質與電壓參數，以及光電流—電壓、飽和電流—輻照度、最大動能—頻率三組實驗結果。
- 黑體輻射：普朗克光譜、峰值波長、總輻射強度及古典紫外災難比較。
- 電磁感應：外加磁場、磁場變化、感應磁場與總磁場的同尺度圖形比較。

### 複習與練習工具

- **物理核心總整理：**統整科學史、重要常數、圖表判讀與跨章節關係。
- **常考情境專欄：**以颱風、空氣阻力落體、地震等八類情境，示範「題幹訊號 → 物理模型 → 第一條關係式」的閱讀方式。
- **知識節點分級題庫：**108 題，分為記憶、理解、應用與分析四個層級，支援單選與多選即時檢核。
- **歷屆試題練習：**收錄 106–115 年學測及 111 年參考試卷，共 124 筆題目資料；一筆資料可對應單題或連續題組，可展開題目裁切圖、作答並查看解析。
- **非選擇題作答：**12 組非選題提供原答案卡作答區，可直接在網頁手寫或作圖，詳解採 CER 結構呈現。其中 10 組來自 111–115 年正式試卷，另 2 組來自 111 年參考試卷；它們均包含在上述 124 筆資料內。
- **五年命題矩陣：**以 111–115 年正式試卷中 71 個主要使用物理概念作答的子題為統計基準，也可選擇納入 106–110 年舊制試題。111 年參考試卷不計入這個 71 題基準。
- **素養包裝破解器：**將跨科或生活化題幹還原成實際使用的物理模型。

## 歷屆題歸類原則

歷屆資料中的「一筆」可能是單題，也可能是共用題幹的連續題組；命題矩陣則以實際參與物理解題的「子題」為單位。跨科題組即使以連續題號收錄，其中不需要物理概念作答的子題也不會納入 71 題統計基準。

每個物理子題再採多節點歸類。若作答時必須同時使用不同章節的公式、判斷或概念，該子題會計入所有參與關鍵推理的節點；只在題幹中出現的名詞、人物或背景情境不會因此列入。

因此，命題矩陣中的各節點題數相加可能大於實際子題總數，這代表跨章整合，而不是重複題目。

## 如何使用這套系統複習

### 第一次建立全科架構

1. 先閱讀「物理核心總整理」，快速建立各章節、重要公式與科學史的整體脈絡。
2. 開啟「五年命題矩陣」，依 111–115 年的出題分布找出高頻節點。
3. 進入節點頁，依序閱讀公式、觀念、關鍵特徵與生活／歷屆應用。
4. 遇到不容易只靠文字理解的概念，再開啟該節點內嵌的互動實驗調整參數觀察。

### 針對弱點補強

1. 從命題矩陣或側欄選擇需要加強的 N01–N28 節點。
2. 先用「記憶、理解」題確認基本定義與公式條件。
3. 再做「應用、分析」題，練習圖表、跨概念與情境轉譯。
4. 回到節點頁的常見陷阱及概念連結，整理錯誤原因，而不只記答案。

### 考前歷屆實戰

1. 在「歷屆試題練習」依年份或知識節點篩選題目。
2. 先完成單選、多選或非選答案卡，再展開答案與詳解。
3. 跨章題可沿題目標示的多個節點回查，確認每一個關鍵判斷都有依據。
4. 非選題先在作答區完成計算或作圖，再以 CER 詳解比較主張、證據與推理是否完整。

### 面對陌生素養情境

先到「常考情境專欄」或「素養包裝破解器」，把題目翻譯成「已知量 → 正在改變的物理量 → 適用模型 → 第一條關係式」，再回題庫或歷屆題驗證。

## 下載與本機使用

本專案是純 HTML、CSS 與 JavaScript，不需要編譯或安裝前端套件。

### 下載 ZIP

1. 前往 [GitHub 專案頁面](https://github.com/ArpowChao/phy-senior)。
2. 點選 **Code → Download ZIP**。
3. 解壓縮後開啟資料夾內的 `index.html`。

### 使用 Git 下載

```bash
git clone https://github.com/ArpowChao/phy-senior.git
cd phy-senior
```

### 直接開啟

開啟 `index.html`，頁面會自動轉至 `gsat_physics_review.html`。

### 使用本機伺服器

若瀏覽器限制本機檔案載入，且電腦已安裝 Python 3，可在專案目錄執行：

```bash
python -m http.server 8000
```

接著前往 `http://localhost:8000/`。數學公式使用 CDN 版 MathJax，字型使用 Google Fonts；完整呈現這兩項外部資源時需要網路連線，除非瀏覽器已有可用快取。

## 主要專案結構

```text
phy-senior/
├─ index.html                  # GitHub Pages 與本機入口
├─ gsat_physics_review.html    # 主要介面、圖表與互動實驗
├─ data/
│  ├─ modulesData.js           # 七大模組的 SOIL 教材內容
│  ├─ questionBank.js          # N01–N28 分級題庫
│  ├─ examPractice.js          # 歷屆題索引、年份與節點歸類
│  └─ examSolutions.js         # 答案、詳解與非選答案卡設定
├─ assets/
│  ├─ exam_crops/              # 歷屆題目裁切圖與答案卡底圖
│  └─ videos/                  # 教學影片資產
├─ images/                     # 早期 111–115 年題圖；現行頁面改用 assets/exam_crops
├─ 108課綱全/                  # 108 課綱試卷與參考資料
└─ 99課綱/                     # 106–110 年舊制試卷與資料
```

修改資料檔後，重新整理瀏覽器即可看到結果；不需編譯或打包。

# 學測物理 SOIL 核心複習智慧系統

## 📌 專案簡介
本專案為基於 **108課綱** 與 **99課綱（舊制延伸）** 必修物理課程設計的線上智慧複習平台。旨在透過結構化的複習模組、歷屆命題矩陣分析、以及具有即時檢核與 CER 科學論證的互動式題庫，協助高中生高效率備考學測自然科（物理）。

---

## 🛠️ 核心架構與名詞解釋

### 1. SOIL 模組化學習法是什麼？
**SOIL** 是本複習系統為每個物理學習主題（Module 1 ~ 7）設計的核心單元結構，由四個學習步驟的英文首字母縮寫組成：
* **S** (Summary / Formulas) **重點整理與核心公式**：條列基礎公式、單位組合與適用情境（如 SI 單位制、因次分析、不確定度）。
* **O** (Overview / Concept Map) **觀念地圖與脈絡**：以關聯圖呈現物理理論與科學史發展脈絡。
* **I** (Insights / Key Features) **關鍵特徵與思維引導**：核心解題特徵解密與「避坑指南」，訓練看見關鍵字聯想對應物理模型（如看到「增強強度而最大動能不變」立刻連結光電效應一對一模型）。
* **L** (Life Connections / Exams) **生活延伸與真題實戰**：結合跨科整合主題與生活科技應用（如悠遊卡與無線充電的電磁感應機制），並搭配歷屆真題與題庫作答。

### 2. CER 科學探究論證是什麼？
**CER** 是本系統引進 108 課綱探究與實作核心素養所採用的科學論證寫作架構，專門為非選擇題與手寫說明題設計。其核心包含：
* **主張 (Claim)**：直接針對問題給出結論或回答。
* **證據 (Evidence)**：列出觀察到的現象、圖表數據或題目給定的已知條件。
* **推理 (Reasoning)**：詳細寫出連結「證據」與「主張」之間的物理定律、理論或學科規律（例如：折射定律定性規則、都卜勒效應、法拉第定律與冷次定律）。
* *本系統在所有歷屆與題庫非選題的詳解中均提供標準的 CER 作答示範，幫助學生組織邏輯清晰的物理答題思維。*

### 3. 歷屆考題統計與五年命題矩陣
系統彙整了近五年（111 年至 115 年學測，以及 111 年參考試卷）共 71 個學測物理子題，將其精確歸類於 27 個核心知識節點（N01 至 N27）中。
* **命題矩陣儀表板**：顯示每個知識節點的歷屆考題總數、分佈年份，幫助學生洞察大考高頻率命題熱點（如電磁感應與冷次定律、能階躍遷、四大基本力等）。

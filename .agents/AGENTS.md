# Physics Handout Information Architecture Rules (物理複習講義編排原則)

為確保物理科複習教材在視覺與認知負荷上符合學生的學習需求，所有知識節點（Knowledge Nodes）的內容生成必須遵循以下四層資訊階級（Information Hierarchy）之編排原則：

## 1. 核心資訊 (Core Principle)
- **呈現位置**：網頁頂部之 **「📖 重點整理與核心公式」** 區塊（對應 formulas 陣列所渲染的白色卡片與灰色 LaTeX 方框）。
- **內容限制**：僅放最核心的定義公式（如科學記號表示法、臨界值）與極精簡的單句定義。
- **嚴禁事項**：請勿將說明文字、應用實例或對照表格塞入白色卡片內，以避免白框視覺過載。

## 2. 輔助資訊 (Explanation of Principles)
- **呈現位置**：白色卡片下方的獨立節區。
- **標題用詞**：**「💡 核心觀念解析」**
- **內容重點**：說明核心公式的物理本質、由來與邏輯推導（例如說明數量級進位臨界值 sqrt(10) 是基於對數尺度的中點）。

## 3. 細節資訊 (Clarifying Details)
- **呈現位置**：輔助資訊下方的獨立節區。
- **標題用詞**：**「🔍 數值與尺度對照」** 或 **「🔍 觀念深度剖析」**
- **內容重點**：使用表格（對照表）、尺度鏈（如原子核到細胞的尺度對比）等結構化工具，將觀念細節以對照方式呈現，使理解清晰。

## 4. 補充資訊 (Application Examples)
- **呈現位置**：節區的最下方。
- **標題用詞**：**「📝 生活應用與估算實戰」** 或 **「📝 跨科整合延伸」**
- **內容重點**：提供具體的生活現象連結與估算步驟（如頭髮生長速率、人體細胞總數的費米估算），展現物理定律的實踐與應用。
